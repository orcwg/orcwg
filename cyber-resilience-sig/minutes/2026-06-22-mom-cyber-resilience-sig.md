### Tech Sovereign Package and Open Source Strategy

* Ref to the available documentation \-  [https://digital-strategy.ec.europa.eu/en/library/communication-european-tech-sovereignty-accompanied-eu-open-source-strategy](https://digital-strategy.ec.europa.eu/en/library/communication-european-tech-sovereignty-accompanied-eu-open-source-strategy)  
* Some of the aspects the Package includes are:  
  * Financial tool to fund maintainers of Open Source components  
  * ENISA should identify critical components to increase resilience when being cut off from parts of the supply chain.  
  * Public administration should contribute to open source as critical infrastructure  
  * It is proposed to create a solutions catalogue that can help to identify professional solutions.  
  * CADA defines different levels of sovereignty for services depending on data management and third country dependencies.  
  * Open Source friendly tenders \- procurement will promote open source

* This is a “declaration of intentions” \- we’ll see how it plays out  
  * There may be additional documents coming
